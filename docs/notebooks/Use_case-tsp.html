In addtion to the base spaghetti requirements (their dependecies), this notebook requires installations of:¶

  • `geopandas <http://geopandas.org>`__

    • $ conda install -c conda-forge geopandas

  • `shapely <https://shapely.readthedocs.io/en/latest/>`__

    • $ conda install -c conda-forge shapely

  • `descartes <https://pypi.org/project/descartes/>`__

    • $ conda install -c conda-forge descartes

  • `matplotlib <https://matplotlib.org>`__

    • $ conda install matplotlib

  • `pulp <https://anaconda.org/conda-forge/pulp>`__

    • $ pip install pulp

  • `watermark <https://github.com/rasbt/watermark>`__

    • $ pip install watermark

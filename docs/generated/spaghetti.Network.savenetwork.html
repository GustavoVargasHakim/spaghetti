spaghetti.Network.savenetwork¶

Network.savenetwork(self, filename)[source]¶

Save a network to disk as a binary file.

Parameters
filenamestr

The filename where the network should be saved. This should be a full path or it will be save in the current directory.

Examples

>>> import spaghetti as spgh
>>> ntw = spgh.Network(examples.get_path('streets.shp'))
>>> ntw.savenetwork('mynetwork.pkl')
